# *基于 URDF 与 Unity3D 的机器人路径规划仿真平台 - Unity端*

## 项目简介

*此项目为毕业设计——《基于 URDF 与 Unity3D 的机器人路径规划仿真平台设计与实现》的一部分。另一部分详见“”项目*

**目标：**构建一个可以与 ROS2 后端进行实时通信的基于 U3D 平台的仿真系统

**职责：**承担主要的仿真任务和路径规划算法的模拟工作，同时可以进行算法效率的可视化对比

**技术栈：**Unity 2022.3.57f1 LTS + C# + TCP 通信

## 环境搭建

- 安装配置 Unity Hub，编辑器选择 Unity 2022 LTS 版本（推荐），选择“3D (Built-In Render Pipeline)“模板
- IDE——Visual Studio Code（推荐）
